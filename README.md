# 🐍 Python 학습 기록 저장소

파이썬의 기초 문법부터 심화 개념까지 차근차근 학습하며 정리하는 공간입니다.  
나중에 실무에서 코드를 참고하거나 복습하기 위해 기록하고 있습니다.

## 🚀 학습 목표
- 파이썬의 핵심 문법을 완벽히 이해하고 활용하기
- 객체지향 프로그래밍(OOP) 개념 익히기
- 깨끗하고 가독성 좋은 코드 작성 연습

### 📂 저장소 구조
학습 순서와 주제에 따라 폴더를 구성하고 있습니다.

- **01_Basic_Syntax**: 변수, 자료형, 기본 연산자 연습
- **02_Control_Flow**: 조건문(if), 반복문(for, while) 제어 흐름 연습
- **03_Functions**: 함수 정의, 매개변수, 반환값 활용
- **04_OOP_Basics**: 클래스, 상속, 생성자 등 객체지향 기초 (예: 은행계좌 시스템)
- **05_Advanced**: 예외 처리, 모듈 및 패키지 활용
